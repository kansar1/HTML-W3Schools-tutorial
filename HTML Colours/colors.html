Colour Values

You can use RGB, HEX, HSL, RGBA and HSLA values as well. The following (div) elements are the same colour

RGB: rgb(255, 99, 71)
HEX: #ff6347
HSL: hsl(9, 100%, 64%)

These two are RGBA and HSLA which add transparency (alpha value)

RGBA: rgba(255, 99, 71, 0.5) (50% transparency)
HSLA: hsla(9, 100%, 64%, 0.5) (50% transparency)
